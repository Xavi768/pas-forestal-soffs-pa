Recomendado
1
Crear cuenta en GitHub
Gratis · 2 minutos · No requiere tarjeta
a
🌐 Ir a GitHub.com
Abrir https://github.com en el navegador y hacer clic en "Sign up" (esquina superior derecha).
b
📧 Registrarse con correo
Ingresar correo electrónico, crear contraseña y elegir un nombre de usuario. Recomendado: usar un nombre profesional como javier-vela-forestal ya que formará parte de la URL pública.
c
✉️ Verificar el correo
Revisar la bandeja de entrada y hacer clic en el enlace de verificación que GitHub envía. Luego iniciar sesión.
💡
La cuenta de GitHub es gratuita y permanente. El plan Free incluye GitHub Pages sin costo. No necesitas pagar nada.
2
Crear el repositorio
El repositorio es la carpeta donde vivirán tus archivos
a
➕ Nuevo repositorio
Una vez dentro de GitHub, hacer clic en el botón verde "New" o en el ícono + de la barra superior → "New repository".
b
🏷️ Configurar el repositorio
Completar el formulario con exactamente estos datos:
# Nombre del repositorio (sin espacios, sin tildes)
Repository name: pas-forestal-soffs-pa

# Descripción opcional
Description: Sistema PAS Forestal - SOFFS-PA Padre Abad 2024

# MUY IMPORTANTE: marcar como Público
Visibility: ⦿ Public (NO Private)

# Marcar esta casilla
☑ Add a README file
c
✅ Crear el repositorio
Hacer clic en el botón verde "Create repository" al final de la página.
3
Subir los archivos HTML
Sin instalar nada — directo desde el navegador
a
📤 Subir archivos
Dentro del repositorio recién creado, hacer clic en "Add file" → "Upload files".
b
🖱️ Arrastrar y soltar los archivos
Arrastrar los tres archivos al área de carga, o hacer clic en "choose your files" para seleccionarlos desde la carpeta donde los guardaste (normalmente el Escritorio o Descargas).
📁
Los archivos están en tu carpeta Descargas o donde los guardaste desde Claude. Subir los 3 archivos juntos de una sola vez.
c
💾 Confirmar la subida
Bajar hasta el final de la página y hacer clic en el botón verde "Commit changes". Los archivos quedarán guardados.
4
Activar GitHub Pages
Este paso publica los archivos en internet con URL permanente
a
⚙️ Ir a Settings
Dentro del repositorio, hacer clic en la pestaña "Settings" (con ícono de engranaje) en la barra superior.
b
📄 Sección Pages
En el menú izquierdo buscar "Pages" (bajo la sección "Code and automation") y hacer clic.
c
🚀 Configurar la fuente
En la sección "Build and deployment", configurar:
Source: Deploy from a branch
Branch: main   /   root
# Luego clic en "Save"
d
⏳ Esperar 1-2 minutos
GitHub tarda entre 1 y 3 minutos en publicar. Recargar la página de Settings → Pages y aparecerá un recuadro verde con el mensaje "Your site is live at:" seguido de la URL pública.
🎯
Tu URL tendrá este formato:
https://TUNOMBREDEUSUARIO.github.io/pas-forestal-soffs-pa/
5
Generar el código QR
Pega tu URL de GitHub Pages y genera el QR al instante
🔗 Generador de QR — PAS Forestal
Una vez que tengas tu URL de GitHub Pages, pégala aquí y el QR se genera automáticamente. Puedes imprimirlo, enviarlo por WhatsApp o incluirlo en tu presentación PowerPoint.
Plantillas de URL — reemplaza TUNOMBRE:
🏠 Página principal
🎙 Agente Expositor
📊 Dashboard
✅ QR generado correctamente
6
Compartir el QR
Formas recomendadas para distribuir el acceso
a
📱 WhatsApp — grupos institucionales
Enviar la URL directamente o el PNG del QR al grupo de trabajo del GORE Ucayali, SOFFS-PA o FEMA. El botón "💬 WhatsApp" del generador abre el chat directamente.
b
📊 En la presentación PowerPoint
Descargar el PNG del QR e insertarlo en la última diapositiva de tu presentación con el texto: "Escanea para acceder al sistema interactivo".
c
🖨️ Imprimir para reuniones presenciales
Imprimir el QR en tamaño A5 o A4 y colocarlo sobre la mesa durante reuniones con autoridades del GORE Ucayali, Fiscalía o gerencias regionales.
🌟
Ventaja de GitHub Pages: El enlace es permanente y gratuito para siempre. Si actualizas los archivos HTML, los cambios se reflejan automáticamente en el mismo QR sin necesidad de generar uno nuevo.
